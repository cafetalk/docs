# 5D T-Rex Persona Ecosystem

The 5D T-Rex Persona Ecosystem is a multi-faceted system designed to transform a static on-chain identity into a dynamic, multi-dimensional digital persona.

## Overview

The 5D Persona Ecosystem enables users to create verifiable, privacy-aware profiles that capture individuals across both Web3 and Web2 surfaces. This creates a living identity that grows with every interaction.

## T-Rex Passport: The Universal On-Chain DID

The T-Rex Passport is a singular, on-chain Decentralized Identifier (DID) for each user. It is a publicly visible smart contract that serves as the nexus of a user's digital life.

## T-Rex Badge System: A Comprehensive Validation Engine

The T-Rex Badge System is a publicly visible, smart contract-based ecosystem that validates and mints badges based on a user's on-chain and off-chain activities.

### Validator Types

#### On-Chain Data Validators

**NFT Validator**
- Supports ERC-721 and ERC-1155 standards
- Validates NFT metadata, transfer history, holding duration
- Tracks DeFi transaction behavior
- Enables badges for NFT collectors and digital art enthusiasts

**Token Validator**
- Supports Ethereum, EVM-compatible chains, and Solana
- Validates wallet balances and transaction history
- Tracks asset holding duration
- Recognizes financial contributions and stability

**Web3 Validator**
- Tracks verifiable on-chain behaviors
- Monitors interactions with designated smart contracts
- Rewards protocol participation and dApp usage

#### ZKTLS Validator

In deep collaboration with Reclaim Protocol, enables private verification of Web2 assets using Zero-Knowledge and TLS proofs:

- **Financial Standing**: CEX total assets, CEX VIP levels
- **Social Influence**: Follower counts on TikTok, Facebook, Instagram, Twitch; Instagram verification
- **Digital Presence**: YouTube subscriber count, GitHub stars
- **Memberships**: Premium memberships for Telegram, ChatGPT, Gemini, Grok, Perplexity, Claude, PS Plus, Xbox Game Pass, Nintendo Switch Online, Spotify, YouTube, Apple Music, Netflix, Amazon Prime
- **Gaming**: Steam game ownership count

#### Ecosystem Validator

Rewards participation within the T-Rex ecosystem:

- **Early Bird Badges**: Rewards for early adopters
- **Event Badges**: Recognizes participation in specific events
- **Student Verification**: Allows verification of student status
- **Custom Validators**: Configurable validators for specific use cases

#### Social Validator

Tracks Web2 social media interactions:

- Account linking
- Follow behavior
- Post activity
- Comprehensive social footprint

## Ecosystem Points

Ecosystem Points are centrally stored within the T-Rex Passport:

- **Badge-Based Points**: Each badge has a corresponding point value
- **Social Task Points**: Different social tasks have specific point values
- **Dynamic Metric**: Reflects user activity, engagement, and contribution
- **Secondary Asset**: Serves as a secondary asset class within the Passport
