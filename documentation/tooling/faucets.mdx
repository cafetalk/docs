# Faucets

Get free test tokens for T-Rex Testnet to develop and test your applications.

## Testnet Faucet

### Access the Faucet

- **Testnet Explorer**: [https://testnet.trex.xyz/](https://testnet.trex.xyz/)
- Look for the "Faucet" button or section

### How to Use

<Steps>
  <Step title="Connect Wallet">
    Connect your wallet (MetaMask recommended) to the testnet
  </Step>
  
  <Step title="Switch to Testnet">
    Ensure your wallet is connected to T-Rex Testnet (Chain ID: 1962)
  </Step>
  
  <Step title="Enter Address">
    Enter or select your wallet address
  </Step>
  
  <Step title="Request Tokens">
    Click "Request Test Tokens" button
  </Step>
  
  <Step title="Wait for Tokens">
    Tokens should arrive within a few seconds
  </Step>
</Steps>

## Test Token Information

- **Token Symbol**: TEST-TREX
- **Network**: T-Rex Testnet
- **Chain ID**: 1962

## Request Limits

### Daily Limits

- **Per Address**: Each address can request a limited amount per day
- **Frequency**: Cooldown period between requests
- **Purpose**: For testing and development only

### Preventing Abuse

<Warning>
The testnet faucet has anti-abuse mechanisms. Please use only for legitimate development and testing purposes.
</Warning>

## Alternative Methods

### Using Testnet Explorer

1. Visit the testnet explorer
2. Navigate to the faucet section
3. Follow the instructions to request tokens

### Developer Program

For developers needing larger amounts of test tokens:

- Contact the development team
- Join the developer Discord
- Apply for developer program access

<Tip>
If you run out of test tokens, you can request more after the cooldown period, or contact the development team for additional allocations.
</Tip>
