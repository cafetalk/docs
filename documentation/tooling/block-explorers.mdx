# Block Explorers

Block explorers allow you to view, search, and analyze transactions, blocks, addresses, and smart contracts on T-Rex Chain.

## Official Block Explorers

### Mainnet Explorer

- **URL**: [https://explorer.trex.xyz/](https://explorer.trex.xyz/)
- **Features**: Complete on-chain data query and analysis

### Testnet Explorer

- **URL**: [https://testnet.trex.xyz/](https://testnet.trex.xyz/)
- **Features**: Testnet data query and debugging

## Using the Explorer

### View Block Information

1. Visit the block explorer
2. Enter a block height or block hash in the search box
3. View block details including:
   - Block timestamp
   - Transaction count
   - Gas used
   - Block reward

### Track Transactions

1. Enter a transaction hash
2. View transaction details:
   - From and to addresses
   - Transaction amount
   - Gas fees
   - Transaction status
   - Event logs

### View Addresses

1. Enter a wallet address
2. View address information:
   - Balance (native token)
   - Transaction history
   - Token balances
   - NFT holdings

### Verify Contracts

1. Navigate to a contract address
2. Click "Verify and Publish"
3. Upload your contract source code
4. The contract will be verified and publicly viewable

## API Access

The block explorer provides REST API for programmatic access:

```bash
# Get block information
curl https://explorer.trex.xyz/api/block/123456

# Get transaction information
curl https://explorer.trex.xyz/api/tx/0x...

# Get address information
curl https://explorer.trex.xyz/api/address/0x...
```
