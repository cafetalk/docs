## Running Your Own Node

### Hardware Requirements

#### Sequencer Node

- **CPU**: 8+ cores
- **RAM**: 32GB+
- **Storage**: 1TB+ SSD
- **Bandwidth**: 100Mbps+

#### RPC Node

- **CPU**: 4+ cores
- **RAM**: 16GB+
- **Storage**: 500GB+ SSD
- **Bandwidth**: 50Mbps+

### Software Requirements

- **OS**: Linux (Ubuntu 20.04+ recommended)
- **Docker**: For containerized deployment
- **Node Software**: T-Rex node client

### Setup Steps

```bash
# Clone node repository
git clone https://github.com/trex-network/node.git
cd node

# Configure node
cp config.example.toml config.toml
# Edit config.toml with your settings

# Start node
./trex-node --config config.toml
```
