# Introduction

T-Rex is an innovative Layer 2 (L2) solution designed to enhance the scalability, security, and interoperability of the Ethereum ecosystem. Built as an EVM-compatible Optimistic Rollup, T-Rex processes transactions off-chain while inheriting the robust security of the Ethereum mainnet.
