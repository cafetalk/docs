# Data & Indexing

Data indexing services provide efficient access to on-chain data, enabling developers to build powerful applications with real-time blockchain information.

## Overview

Data indexing services on T-Rex Chain provide:

- **Fast Queries**: Optimized data retrieval for applications
- **Historical Data**: Access to complete blockchain history
- **Real-time Updates**: Live data synchronization
- **Rich Metadata**: Enhanced data with additional context

### Application Development

### Data Analysis

- Historical transaction analysis
- Token flow tracking
- Protocol usage metrics
- User behavior insights

### API Integration

Access indexed data through REST APIs and GraphQL:

```javascript
// Example: Query token transfers
const response = await fetch('https://api.trex.xyz/v1/tokens/transfers', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    address: '0x...',
    token: '0x...',
    fromBlock: 0,
    toBlock: 'latest'
  })
});
```

## Indexing Providers

### Official Indexing Service

T-Rex provides official indexing services for core on-chain data:

- **Block Explorer API**: Access data through the block explorer
- **RPC Endpoints**: Direct RPC calls for indexed data
- **WebSocket Streams**: Real-time data subscriptions

### Third-Party Providers

Various third-party indexing services may be available:

- **The Graph**: Subgraph indexing for T-Rex Chain
- **Custom Indexers**: Custom indexing solutions
- **Data Providers**: Specialized data services

## Query Examples

### Get Token Balance

```javascript
const balance = await indexer.getTokenBalance(
  '0x...', // address
  '0x...', // token contract
  'latest' // block number
);
```

### Get Transaction History

```javascript
const transactions = await indexer.getTransactions({
  address: '0x...',
  fromBlock: 0,
  toBlock: 'latest',
  limit: 100
});
```

### Subscribe to Events

```javascript
indexer.subscribe('Transfer', {
  address: '0x...',
  from: '0x...',
  to: '0x...'
}, (event) => {
  console.log('Transfer event:', event);
});
```
