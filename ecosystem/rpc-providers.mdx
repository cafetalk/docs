# RPC Providers

RPC (Remote Procedure Call) providers offer access to T-Rex Chain through JSON-RPC endpoints, enabling applications to interact with the blockchain.

## Public RPC Endpoints

### Mainnet

```
HTTPS: https://rpc.trex.xyz/
WSS: wss://rpc.trex.xyz/ (if available)
```

### Testnet

```
HTTPS: https://testnetrpc.trex.xyz
WSS: wss://testnetrpc.trex.xyz (if available)
```

## RPC Provider Options

### Official RPC Endpoints

T-Rex provides official public RPC endpoints:

- **Free Tier**: Basic rate limits for development
- **Reliable**: High uptime and availability
- **Standard Methods**: All standard Ethereum JSON-RPC methods

### Third-Party Providers

Various third-party providers offer enhanced RPC services:

#### Infrastructure Providers

- **Alchemy**: Enterprise-grade RPC infrastructure
- **Infura**: Reliable RPC endpoints with analytics
- **QuickNode**: High-performance RPC nodes
- **Ankr**: Distributed RPC infrastructure

#### Features

- **Enhanced APIs**: Additional endpoints and features
- **Higher Rate Limits**: Increased request limits
- **Analytics**: Usage statistics and monitoring
- **Support**: Technical support and documentation

## Using RPC Endpoints

### Basic Usage

```javascript
const { ethers } = require('ethers');

// Connect to T-Rex Mainnet
const provider = new ethers.providers.JsonRpcProvider('https://rpc.trex.xyz/');

// Get block number
const blockNumber = await provider.getBlockNumber();
console.log('Current block:', blockNumber);
```

### With API Key

Some providers require API keys:

```javascript
const provider = new ethers.providers.JsonRpcProvider(
  'https://eth-mainnet.g.alchemy.com/v2/YOUR_API_KEY'
);
```

### WebSocket Connection

```javascript
const { ethers } = require('ethers');

const wsProvider = new ethers.providers.WebSocketProvider('wss://rpc.trex.xyz/');

wsProvider.on('block', (blockNumber) => {
  console.log('New block:', blockNumber);
});
```

## Rate Limits

### Public Endpoints

- **Free Tier**: 100 requests/minute
- **Rate Limit Headers**: Check response headers for limits
- **Throttling**: Requests may be throttled if exceeded

### Paid Plans

- **Higher Limits**: Increased request limits
- **Priority Processing**: Faster request handling
- **SLA Guarantees**: Service level agreements
