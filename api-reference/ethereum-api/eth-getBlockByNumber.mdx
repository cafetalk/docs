# eth_getBlockByNumber

Returns information about a block by block number

<ParamField body="body" type="object">
  <ParamField body="jsonrpc" type="string" default="2.0" required>
    JSON-RPC version
  </ParamField>
  <ParamField body="method" type="string" default="eth_getBlockByNumber" required>
    Method name
  </ParamField>
  <ParamField body="id" type="integer" default="1" required>
    Request identifier
  </ParamField>
  <ParamField body="params" type="array" required>
    <ParamField body="0" type="string" required>
      Block number (hex) or tag ("latest", "earliest", "pending")
    </ParamField>
    <ParamField body="1" type="boolean" default="false">
      If true, returns full transaction objects; if false, returns transaction hashes
    </ParamField>
  </ParamField>
</ParamField>

<ResponseField name="result" type="object">
  Block object with transaction data
</ResponseField>

## Example Request

<CodeGroup>

