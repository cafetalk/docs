# eth_getBalance

Returns the balance of an account at a given block

<ParamField body="body" type="object">
  <ParamField body="jsonrpc" type="string" default="2.0" required>
    JSON-RPC version
  </ParamField>
  <ParamField body="method" type="string" default="eth_getBalance" required>
    Method name
  </ParamField>
  <ParamField body="id" type="integer" default="1" required>
    Request identifier
  </ParamField>
  <ParamField body="params" type="array" required>
    <ParamField body="0" type="string" required>
      Address to check balance for (20-byte hex string)
    </ParamField>
    <ParamField body="1" type="string">
      Block number or tag (default: "latest")
    </ParamField>
  </ParamField>
</ParamField>

<ResponseField name="result" type="string">
  Balance in wei as hexadecimal string
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://rpc.trex.xyz/ \
  --header 'Content-Type: application/json' \
  --data '{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "eth_getBalance",
  "params": [
    "0xbbB216C52f5B9095391f463AdC438d65a8aC2242",
    "latest"
  ]
}'
```

```javascript JavaScript
const response = await fetch('https://rpc.trex.xyz/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    jsonrpc: '2.0',
    id: 1,
    method: 'eth_getBalance',
    params: [
      '0xbbB216C52f5B9095391f463AdC438d65a8aC2242',
      'latest'
    ]
  })
});

const data = await response.json();
const balance = BigInt(data.result);
console.log('Balance:', balance.toString(), 'wei');
```

</CodeGroup>

## Example Response

```json
{
  "jsonrpc": "2.0",
  "result": "0x1bc16d674ec80000",
  "id": 1
}
```

<Tip>
The result is in wei (smallest unit). To convert to TREX: `balance / 10^18`
</Tip>
